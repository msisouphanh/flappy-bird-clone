## Flappy Bird Clone
A Flappy Bird Clone game.

## Technologies
- `Python`
- `Pygame`

## Installation`
1. Clone the repository
2. Install the Pygame library by using `py -m pip install pygame`
3. Compile `fappy_bird.py`
